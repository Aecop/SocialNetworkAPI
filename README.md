## Installation
To get started clone this repository using
<br>
```terminal
git clone [https://github.com/Aecop/E-CommerceBackEnd.git](https://github.com/Aecop/SocialNetworkAPI)
```

Then, 
```terminal
npm install
``` 

Start the server
```terminal
npm start
```
Navigate to Insomia
